# SIEM Plataform
---
Siem mean Security Information and Event Management, It is a security solution that centralizes, collects, and analyzes data from various network sources in real time.

## Why
---
Analyzing logs is very tedious and time-consuming, especially when the logs are repetitive and low-risk; however, these logs often hide relevant information, such as system errors, database connection errors, routes failing at specific times, etc.
